# 🔨 Finance Forge API

Бэкенд-приложение на **FastAPI** и **SQLAlchemy 2.0** для учета личных доходов и накопления на финансовые цели.

## Технологии
* Python 3.12+
* FastAPI
* SQLAlchemy 2.0 (ORM)
* SQLite

## Как запустить на работе:
1. Установи зависимости: `pip install fastapi uvicorn sqlalchemy`
2. Запусти сервер: `uvicorn main:app --reload`
3. Открой интерактивную документацию: `http://127.0.0.1:8000/docs`
